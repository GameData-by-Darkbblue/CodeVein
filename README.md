# CodeVein
## 内容说明
本人整理的相关资料，划分为多个部分：  
**基本机制**：一些基础的机制。  
**流程**：既有整体流程与地图流程作为指南使用，也有路线规划与重要物品位置的说明。  
**打法**：从配装思路角度进行分类，以及根据boss进行分类。  
**Data**：取自贴吧/群整理的炼血与boss数据，并含有公式说明。制作者参见PDF文件内容。  
**剧情**：剧情整理。
